# Website Builder
The base `build.py` and its library, along with the `_layouts`

## Arch
1. `prebuild.py` fetches `WebsiteBuilder` repo
2. In case of version compatibility of `build.py` and the data itself
3. `build.py` runs in the new `root` folder with its corresponding data
4. The final website is runned on its new repo
